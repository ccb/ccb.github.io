Introduction to Computer Networking

25 January 2014 - 8 April 2014 This class was the easily the best organized of any MOOC class I've taken. The course structure was heavily based on a CS 144 which is the introductory networking class taught at Stanford. Each of the units was full of content and had complete and comprehensive quizzes. The topics ranged from routing to NAT traversal to low-level implementation details. [SOA]
